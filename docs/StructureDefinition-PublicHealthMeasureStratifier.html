StructureDefinition: PublicHealthMeasureStratifier

Profile Saner Public Health Measure Stratifier

A stratifier is effecitively a mixin that can be used with an existing measure to add stratification detail to that measure.

The official URL for this profile is: 

http://hl7.org/fhir/us/saner/StructureDefinition/PublicHealthMeasureStratifier
